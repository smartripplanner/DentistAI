Google Sheets Integration

Connect DentalAI directly to a Google Spreadsheet. Appointments, qualified leads, and triage alerts will sync automatically.

Disconnected (Local Storage Active)
Triggers email alerts on Handoff Escalations and Emergencies.

Setup Instructions

  1. Create a blank Google Sheet.
  2. Navigate to Extensions ➔ Apps Script.
  3. Paste the code from the file google_apps_script.js into the script editor.
  4. Click Deploy ➔ New deployment. Select type Web app.
  5. Configure: Execute as: "Me" and Who has access: "Anyone".
  6. Click Deploy, authorize access, and copy the generated Web app URL.
  7. Paste the URL on the left and click Save & Connect. The script will automatically create the sheets and headers!

💡 Why aren't emails or calendar events working?

  • Authorization Blocked (Calendar/Gmail): You must grant permissions inside Google. In the Apps Script Editor, select triggerAuthorization from the dropdown in the top toolbar and click Run. This will force Google to show the authorization prompt. Click "Review Permissions" and grant access.
  • Apps Script Deployments cache aggressively: Whenever you edit the script, you MUST click Deploy ➔ Manage Deployments ➔ Edit (Pencil) ➔ select Version: "New version" ➔ Deploy. Simply clicking "Save" does NOT run your new changes!
  • Incorrect permissions: Ensure your web app deployment is configured with Execute as: "Me" and Who has access: "Anyone".
  • Google Executions History: In the Apps Script editor, open the Executions tab from the sidebar. You can inspect any failed run to see Google's internal logs and exceptions.
